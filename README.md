# tlrunner
TLauncher for Arch Linux

## Description
A dynamic bash launcher for TLauncher

# Installation

Copy the GitHub repository (or download it manually)
```bash
git clone https://github.com/enelminun/tlrunner.git
cd tlrunner
makepkg -si
```
